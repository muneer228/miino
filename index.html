Stack and direction

The stack is intentionally small, inspectable, and modular: local-first when needed, cloud-hosted when it makes sense, and ready for more public-facing apps.

System Arch Linux, local development, small services, and an emphasis on direct control over tooling and deployment.
Network Cloudflare-backed routing for public access, with the flexibility to split static hosting from laptop-hosted apps.
Design Minimal but expressive interfaces with strong emphasis on readability, visual polish, and practical information density.
Direction A home for project pages, hosted demos, publications, notes, downloads, and future tools under one clear brand.
